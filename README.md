# VAFLYA — Информационная система складского учёта

Веб-приложение для учёта товаров, остатков и складских операций.

## Технологический стек

Frontend: React 18. Backend: Node.js + Express. База данных: SQLite (файл backend/warehouse.db).

## Запуск

1. Бэкенд: cd backend → npm install → npm start. API будет доступен на http://localhost:3001.
2. Фронтенд (в новом терминале): cd frontend → npm install → npm start. Приложение откроется на http://localhost:3000.

## Функциональность

Главная: сводка (кол-во товаров, остатки, операции, низкий запас), последние операции. Товары: добавление, редактирование, удаление, артикул, категория, единица измерения, минимальный остаток. Остатки: текущие количества по всем позициям, статус (в наличии / мало / нет), местоположение. Операции: журнал поступлений и списаний, фильтр по типу и товару, форма создания операции. Категории: справочник категорий товаров, добавление и удаление.

## API эндпоинты

GET /api/stats — статистика для главной страницы. GET и POST /api/products — список товаров и создание. PUT и DELETE /api/products/:id — обновление и удаление товара. GET /api/stock — текущие остатки. PUT /api/stock/:id — обновить местоположение. GET и POST /api/operations — журнал и создание операции. GET, POST и DELETE /api/categories — управление категориями.
